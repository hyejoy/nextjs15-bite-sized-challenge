# Section 03-02.네비게이팅

## Link 컴포넌트 이용하여 CSR 방식 사용

```ts
import Link from "next/link";

export default function App({ Component, pageProps }: AppProps) {
  return (
    <>
      <header>글로버 헤더</header>
      <Link href={"/"}>index</Link>
      &nbsp;
      <Link href={"/search"}>search</Link>
      &nbsp;
      <Link href={"/book/1"}>book/1</Link>
      &nbsp;
      <Component {...pageProps} />
    </>
  );
}
```

## 프로그래메틱하게 페이지 이동 ≒ 함수 내부에서 페이지 이동

- useRouter의 router객체의 **`push()`** 메서드 활용하기

```ts
import { useRouter } from "next/router";
export default function App({ Component, pageProps }: AppProps) {
  const router = useRouter();
  const onClickButton = () => {
    router.push("/test");
  };
  return (
    <>
      <div>
        <button onClick={onClickButton}>/test 페이지이동</button>
      </div>
    </>
  );
}
```

- router에는 push메서드 말고도 **`replace()`** 함수
  └── 뒤로가기 방지하면서 페이지 이동
- **`back`** 함수도 존재함
  └── 뒤로 가기를 시키는 메서드
